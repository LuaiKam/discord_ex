    Discord Elixir v1.0.0 DiscordElixir.Client

    Connect to Discord to recieve and send data in realtime You shouldn’t be using this directly. You should shold pass it to a handler.

    Examples

    token = "<your-token>"
DiscordElixir.Client.start_link(%{token: token, handler: DiscordElixir.EchoBot})
#=> {:ok, #PID<0.178.0>}
